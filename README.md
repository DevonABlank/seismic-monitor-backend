# CommandValidator

**Clue-Less** · Team Advancement in Motion · Owner: Devon Blank (DB)

Validates incoming player commands against the current game state before anything is written to the event log.

---

## Setup & Run

**Requires Python 3.10+**

```bash
# 1. Clone the repo
git clone <repo-url>
cd clue-less

# 2. Install dependencies (none yet)
pip install -r requirements.txt

# 3. Run
python command_validator.py
```
